Coder Notes

Coder Notes is a web application that combines Evernote, Sublime and repl.it together. Once logged in, users can manage their notes into different notebooks by topic. It is not only able to highlight code syntax, but also run the code on the same interface. Moreover, if a user finds a useful tutorial or website, they can highlight it and save it to their notes using Coder Notes Chrome extension. Coder Notes is deployed through Heroku cloud platform and Coder Notes Chrome extension is available in Chrome web store.

Technologies: MEAN Stack, Markdown, Highlight.js, Tonic and Chrome Extension

Talking Recipes

Talking recipes is an interactive recipe site. For every single recipe, the application generates an audio file for the recipe instrution and play it step by step.The corresponding text is also hilighted while audio is playing. It recognizes user's vioce to control the audio play and/or to search recipes.

Technologies: MEAN stack, recipe API, Web Speach API (voice recognition + voice generator)
